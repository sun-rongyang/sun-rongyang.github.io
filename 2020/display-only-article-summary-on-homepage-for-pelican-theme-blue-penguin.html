Display only article summary on homepage for Pelican theme blue-penguin

In the original blue-penguin theme, the whole article will be displayed on the homepage. If the article is long, your homepage will look verbosely. You can slightly modify the theme to display the Summary of an article only.

Note that I use the version of the blue-penguin is 1.0.0

Only display the summary

Let us suppose your website project root directory is proj-root. Go to the proj-root and

> cp themes/blue-penguin/templates/article_stub.html themes/blue-penguin/templates/article_stub_home.html

Then Open the themes/blue-penguin/templates/article_stub_home.html and modify the article.content to article.summary.

Then Open the themes/blue-penguin/templates/index.html and modify

{%   include "article_stub.html" %}

to

{%   include "article_stub_home.html" %}

Now your homepage will only display the article summary.

Add a hyperlink button to jump to the article

It is very convenient to add a hyperlink to jump to the article full text under the article summary on the homepage. Add the following code to article_stub_home.html under the line {{ article.summary }} in your theme template directory.

<p>
    <div class="more">
        <a href="{{ SITEURL }}/{{ article.url }}">More</a>
    </div>
</p>

Then set some format for this in the theme CSS file themes/blue-penguin/static/css/main.css. Add these settings

div.more a {
    color: #BBB;

    font-family: sans-serif;
    font-weight: bold;
}

Finally regenerate your website add deploy it. For the real effect, you can see the homepage of this blog.
