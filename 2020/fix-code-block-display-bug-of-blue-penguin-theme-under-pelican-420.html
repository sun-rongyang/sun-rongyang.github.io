Fix code block display bug of blue penguin theme under Pelican 4.2.0

I do not know which is the starting version, but the Markdown code block will be parsed as

<div class="highlight">
  <code>
    your code
  </code>
</div>

under Pelican 4.2.0. But the blue-penguin v1.0.0 seems not handle this in its CSS file. So the code block can not be formatted rightly. Here you can slightly modify the CSS file to fix this bug.

Open <your_blue_penguin_theme_root>/static/css/main.css, add a new block for .highlight code

.highlight code {
    background-color: #002b36;
    color: #93a1a1;
}

Then change the white-space in the code block

white-space: pre;

to fix the no line breaking issue.
